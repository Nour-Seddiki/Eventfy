# Eventfy
🎉 Eventfy – Event Management Platform

Eventfy is a modern event management platform that allows organizers to create, manage, and monitor events efficiently. It provides a complete digital solution for handling registrations, ticketing, QR-based access control, and attendee engagement.

The platform is designed to support physical, virtual, and hybrid events with a scalable backend architecture.

🚀 Features :

👤 User Authentication (JWT-based)

📅 Event Creation & Management

🎟 Ticket Registration System

📧 Email Verification & OTP Support

🔐 QR Code Generation for Event Access

📝 Event Reviews & Ratings

📊 Admin Dashboard Support (optional extension)

🔄 RESTful API built with FastAPI

🏗 Tech Stack :

Backend: FastAPI

Database: PostgreSQL / SQLite

ORM: SQLAlchemy

Authentication: JWT

Email Service: SMTP

QR Code Generation: UUID-based system

🎯 Project Goal:

Eventfy aims to simplify event organization by providing a secure, scalable, and easy-to-use backend system that can integrate with web or mobile frontends.
